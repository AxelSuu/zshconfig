# zshrc

Personal zsh config with sane defaults and terminal toys.

## Features

- **Prompt** — Starship
- **Plugins** — autosuggestions, syntax highlighting, zoxide, fzf
- **CLI tools** — eza, bat, nvim (with fallbacks if not installed)
- **History** — 100k lines, deduped, shared across terminals
- **NVM** — lazy-loaded (no startup delay)
- **SSH agent** — one persistent agent per user

## Commands

| Command | Description |
|---|---|
| `väder` | Linköping weather forecast |
| `stock <ticker>` | Stock chart in terminal |
| `rate <coin>` | Crypto/currency rates |
| `tmap` | ASCII world map |
| `qr <url>` | QR code in terminal |
| `yeet [msg]` | git add · commit · push |
| `up <n>` | Go up N directories |
| `ff <name>` | Fuzzy find files |
| `fe` | Fuzzy find and open in editor |
| `take <dir\|url>` | mkdir+cd or git clone+cd |
| `extract <file>` | Extract any archive |
| `timer [min]` | Countdown timer (default 25 min) |
| `transfer <file>` | Upload file, get shareable URL |
| `cheat <cmd>` | Cheatsheet for any command |
| `news [topic]` | Terminal news |

## Git shortcuts

`gs` · `ga` · `gc` · `gp` · `gl` · `gd` · `co`

## Directory bookmarks

`~code` · `~proj` · `~dl` — edit in `.zshrc` to match your paths
